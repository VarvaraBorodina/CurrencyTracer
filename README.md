# TModsen Currency Tracker

## Функционал:

- Конвертер валют;
- График изменения валюты в течение месяца;
- Смена темы приложения;
- Elastic search - поиск с выпадающим списком.

## Используемые технологии

- **_node.js_**
- **_babel_**
- **_axios_**
- **_chart-js_**
- **_mapbox-gl_**
- **_eslint_**
- **_yarn_**
- **_react_**
- **_webpack_**
- **_prop-types_**
- **_redux_**
- **_cypress_**
- **_jest_**
